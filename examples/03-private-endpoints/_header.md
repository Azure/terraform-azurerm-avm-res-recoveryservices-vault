# Private Endpoints Example

This example deploys a Recovery Services Vault with private connectivity for Azure Backup and Azure Site Recovery.

## What this example shows

- Creating private endpoints for vault subresources
- Associating private DNS zones for Backup, Site Recovery, blob, and queue traffic
- Using managed identities together with private network access patterns
- A private-network design suitable for environments that restrict public access paths
